게시판 만들기 (Spring boot)
=


Spring boot3 으로 만든 게시판입니다.

### 사용기술
  - SpringBoot 3
  - JAVA 17
  - Thymeleaf
  - mariaDB
  - JPA
  - 

    
### 🛠️구현기능

- **게시판 기능**
    - 모든 게시글 및 특정 게시글 조회
    - 게시글 검색 (제목, 내용, 작성자)
    - 게시글 작성 [회원]
    - 게시글 수정 [회원, 게시글 작성자]
    - 게시글 삭제 [회원, 게시글 작성자]
    - 게시글 파일 업로드 및 다운로드 [회원, 게시글 작성자] < 작업중
    - 게시글 답글 작성 [회원] < 작업예정


- **회원 기능 + JWT + OAuth2**
    - 회원가입
    - 로그인/로그아웃
